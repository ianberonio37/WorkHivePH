How this report is built
  • Plants in network: number of distinct hives contributing anonymised data to this report.
  • Top failure cause: the most-common root cause across all hives in the period.
  • Report freshness: days since the report was generated (refreshes monthly).
  • Verdict: green when report has 3+ plants and was generated in last 45 days; empty otherwise.
