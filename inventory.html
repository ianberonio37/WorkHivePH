How these counts are computed
  • Out of stock: parts with qty_on_hand <= 0.
  • Low stock: parts with 0 < qty_on_hand <= min_qty (the reorder threshold).
  • Pending approval: parts a worker added in this hive that need a supervisor sign-off before Use / Restock unlocks.
  • Verdict: red if any out-of-stock for critical parts; amber if any low-stock OR pending; green when fully stocked.
