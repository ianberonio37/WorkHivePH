How integrations are categorised
  • Active integrations: integration_configs rows with enabled = true and a sync in the last 7 days.
  • Stale syncs: enabled integrations whose last_sync_at is older than 7 days — usually an auth or endpoint issue.
  • Disabled: integrations turned off but still in the config table.
  • Verdict: red if any stale; amber if any disabled OR no integrations at all; green when all active integrations are fresh.

The 3 tabs below (Import File / Live Sync / API Keys) let you configure each integration type.
